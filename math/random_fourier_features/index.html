Random Fourier Features

Random Fourier Features

An interactive explainer — Rahimi & Recht, NeurIPS 2007

The problem: Kernel methods — SVMs, Gaussian Processes, Kernel PCA — are powerful nonlinear models that implicitly compute inner products in infinite-dimensional feature spaces. But they require building an $n \times n$ kernel matrix, costing $O(n^3)$ to train. At $n = 10^5$ this is infeasible.

The solution: For any shift-invariant kernel, construct a random $D$-dimensional feature map $z: \mathbb{R}^d \to \mathbb{R}^D$ such that the dot product approximates the kernel:

$$k(\mathbf{x}, \mathbf{y}) \approx z(\mathbf{x})^\top z(\mathbf{y}), \qquad z(\mathbf{x}) = \frac{1}{\sqrt{D}} \begin{bmatrix} \sqrt{2}\cos(\boldsymbol{\omega}_1^\top \mathbf{x} + b_1)\\ \vdots\\ \sqrt{2}\cos(\boldsymbol{\omega}_D^\top \mathbf{x} + b_D) \end{bmatrix}$$

Training a linear model on these features approximates the kernel machine, reducing cost to $O(nD)$ — linear in $n$ when $D \ll n$.

The central identity driving everything: $$k(\mathbf{x} - \mathbf{y}) = \underbrace{\mathbb{E}_{\boldsymbol{\omega} \sim p(\boldsymbol{\omega})}\!\left[\cos(\boldsymbol{\omega}^\top(\mathbf{x}-\mathbf{y}))\right]}_{\text{Bochner's theorem (1932)}} \approx \underbrace{z(\mathbf{x})^\top z(\mathbf{y})}_{\text{D-sample Monte Carlo}}$$
🏆 NeurIPS 2017 Test of Time Award — one of ML's most cited papers.

🔑 Kernel Methods

What is a kernel?

A kernel function $k : \mathcal{X} \times \mathcal{X} \to \mathbb{R}$ measures similarity between two data points. The fundamental property:

$$k(\mathbf{x}, \mathbf{x}') = \langle \phi(\mathbf{x}), \phi(\mathbf{x}') \rangle_{\mathcal{H}}$$

where $\phi: \mathcal{X} \to \mathcal{H}$ is a (possibly implicit) map into a Hilbert space $\mathcal{H}$. You compute the similarity without ever constructing $\phi$.

Validity — Positive Semi-Definiteness (PSD)

The Gram matrix $K \in \mathbb{R}^{n \times n}$ with $K_{ij} = k(x_i, x_j)$ must satisfy:

$$\mathbf{c}^\top K \mathbf{c} \geq 0 \quad \forall \mathbf{c} \in \mathbb{R}^n$$

Mercer's Theorem (1909)

A continuous symmetric function is a valid kernel iff its Gram matrix is PSD for every finite set of points. It admits a spectral expansion:

$$k(\mathbf{x}, \mathbf{x}') = \sum_{i=1}^\infty \lambda_i \psi_i(\mathbf{x})\psi_i(\mathbf{x}'), \quad \lambda_i \geq 0$$

This decomposition is the formal bridge to Random Fourier Features.

The Kernel Trick

Many ML algorithms depend only on inner products between data points. Replacing $\langle x_i, x_j \rangle$ with $k(x_i, x_j)$ lifts the algorithm into a high-dimensional feature space at cost $O(d)$ per evaluation.

Concrete example — degree-2 polynomial kernel

Let $\mathbf{x} = (x_1, x_2)$. Define the explicit feature map:

$$\phi(\mathbf{x}) = (x_1^2,\; x_2^2,\; \sqrt{2}\,x_1 x_2)$$

Then:

$$\langle \phi(\mathbf{x}), \phi(\mathbf{x}') \rangle = x_1^2{x_1'}^2 + x_2^2{x_2'}^2 + 2x_1x_2x_1'x_2' = (\mathbf{x} \cdot \mathbf{x}')^2$$

A 3D inner product via one 2D dot product. For degree $d$ in $p$ dimensions, the feature space has $O(p^d)$ dimensions but the kernel costs $O(p)$.

The RBF kernel takes this to the extreme: its implicit feature space is infinite-dimensional, yet $k(\mathbf{x},\mathbf{x}') = \exp(-\|\mathbf{x}-\mathbf{x}'\|^2/2\sigma^2)$ evaluates in $O(d)$.

📐 Bochner's Theorem

The theoretical foundation of Random Fourier Features

Shift-Invariant Kernels

A kernel is shift-invariant (stationary) if it depends only on the difference between its inputs:

$$k(\mathbf{x}, \mathbf{y}) = k(\mathbf{x} - \mathbf{y})$$
⚠️ Notation clarification — why does $k$ appear with both one and two inputs?

This is a deliberate notation overload — the same symbol $k$ is used for two related but distinct functions:

  • Left side $k(\mathbf{x},\mathbf{y})$ — kernel as a function of two inputs
  • Right side $k(\mathbf{x}-\mathbf{y})$ — kernel as a function of one input (the difference vector)

The statement says the two-argument kernel's value depends only on the difference $\mathbf{x}-\mathbf{y}$, not on $\mathbf{x}$ and $\mathbf{y}$ individually. So you can define a single-argument function taking the difference as input.

RBF kernel — both forms are the same thing:

$$k(\mathbf{x}, \mathbf{y}) = \exp\!\left(-\frac{\|\mathbf{x} - \mathbf{y}\|^2}{2\sigma^2}\right) = \exp\!\left(-\frac{\|\boldsymbol{\Delta}\|^2}{2\sigma^2}\right) = k(\boldsymbol{\Delta}), \quad \boldsymbol{\Delta} = \mathbf{x} - \mathbf{y}$$

$\mathbf{x}$ and $\mathbf{y}$ only ever appear as $\mathbf{x}-\mathbf{y}$, so the single-argument form is equivalent.

Polynomial kernel (not shift-invariant):

$$k(\mathbf{x}, \mathbf{y}) = (\mathbf{x} \cdot \mathbf{y})^2$$

This depends on the dot product, not the difference, so it cannot be written as $k(\mathbf{x}-\mathbf{y})$.

Intuition: shift-invariance means the kernel behaves like a distance meter — it only cares how far apart two points are, not where they sit in space.

Bochner's Theorem (1932)

A continuous, bounded, shift-invariant function $k(\boldsymbol{\Delta})$ on $\mathbb{R}^d$ is positive semi-definite iff it is the Fourier transform of a non-negative measure $p(\boldsymbol{\omega})$: $$k(\mathbf{x} - \mathbf{y}) = \int_{\mathbb{R}^d} p(\boldsymbol{\omega})\, e^{i\boldsymbol{\omega}^\top(\mathbf{x}-\mathbf{y})}\, d\boldsymbol{\omega}$$

When $k(\mathbf{0})=1$, the measure $p(\boldsymbol{\omega})$ is a proper probability distribution — the spectral density of the kernel.

For real, symmetric $p(\boldsymbol{\omega})$ the imaginary parts cancel, giving the cosine form:

$$k(\mathbf{x} - \mathbf{y}) = \mathbb{E}_{\boldsymbol{\omega} \sim p}\!\left[\cos\!\left(\boldsymbol{\omega}^\top(\mathbf{x} - \mathbf{y})\right)\right]$$
Key implication: the kernel is an expectation of cosines under the spectral distribution. Sample from $p(\boldsymbol{\omega})$, average the cosines — you get the kernel. That is exactly RFF.

Audio analogy

Just as any audio signal decomposes into pure sinusoids via its Fourier spectrum, any stationary kernel decomposes into sinusoidal basis functions weighted by $p(\boldsymbol{\omega})$. The spectral density is the kernel's frequency fingerprint.

🎲 The RFF Algorithm

Step-by-step derivation

Step 1 — Bochner in complex form

$$k(\mathbf{x}-\mathbf{y}) = \mathbb{E}_{\boldsymbol{\omega}\sim p} \left[e^{i\boldsymbol{\omega}^\top \mathbf{x}}\, e^{-i\boldsymbol{\omega}^\top \mathbf{y}}\right]$$

Step 2 — Simplify to real cosines

For symmetric $p(\boldsymbol{\omega})$, imaginary parts cancel:

$$k(\mathbf{x}-\mathbf{y}) = \mathbb{E}_{\boldsymbol{\omega}\sim p} \left[\cos\!\left(\boldsymbol{\omega}^\top(\mathbf{x}-\mathbf{y})\right)\right]$$

Step 3 — Random phase trick  ($b \sim U[0,2\pi]$)

$$\mathbb{E}_{b}\!\left[2\cos(\boldsymbol{\omega}^\top\mathbf{x}+b)\cos(\boldsymbol{\omega}^\top\mathbf{y}+b)\right] = \cos(\boldsymbol{\omega}^\top(\mathbf{x}-\mathbf{y}))$$

Proof: expand via $\cos\alpha\cos\beta = \tfrac{1}{2}[\cos(\alpha-\beta)+\cos(\alpha+\beta)]$. The $\cos(\alpha+\beta)$ term integrates to zero over $U[0,2\pi]$. ∎

⚠️ Why is this trick needed at all?

The core problem: factorization. From Bochner we have $k(x-y) = \mathbb{E}_\omega[\cos(\omega^\top(x-y))]$, but $\cos(\omega^\top(x-y))$ is a function of both $x$ and $y$ entangled together. An explicit feature map requires a dot product:

$$k(x,y) \approx z(x)^\top z(y) = \sum_j \underbrace{z_j(x)}_{\text{only } x} \cdot \underbrace{z_j(y)}_{\text{only } y}$$

So we need to split $\cos(\omega^\top(x-y))$ into a product of a function of $x$ alone and a function of $y$ alone. There are two ways to do this:

Option A — sin/cos decomposition (no random phase needed):

Use the identity $\cos(a-b) = \cos a \cos b + \sin a \sin b$:

$$\cos(\omega^\top(x-y)) = \underbrace{\cos(\omega^\top x)}_{\text{only }x}\underbrace{\cos(\omega^\top y)}_{\text{only }y} \;+\; \underbrace{\sin(\omega^\top x)}_{\text{only }x}\underbrace{\sin(\omega^\top y)}_{\text{only }y}$$

Feature map: $z(x) = \tfrac{1}{\sqrt{D}}[\cos(\omega_1^\top x),\, \sin(\omega_1^\top x),\, \ldots] \in \mathbb{R}^{2D}$. This works — but produces a $2D$-dimensional map.

Option B — random phase trick (what Rahimi & Recht use):

$$\mathbb{E}_{b \sim U[0,2\pi]}\!\left[2\cos(\omega^\top x + b)\cdot\cos(\omega^\top y + b)\right] = \cos(\omega^\top(x-y))$$

Feature map: $z(x) = \tfrac{1}{\sqrt{D}}[\sqrt{2}\cos(\omega_1^\top x + b_1),\, \ldots] \in \mathbb{R}^{D}$. Also works — and is only $D$-dimensional (half the size).

The random phase trick is not strictly necessary — Option A (sin/cos) is equally valid and arguably more intuitive. Option B was chosen because it gives a more compact feature vector (D vs 2D dimensions) using only cosines. Both are unbiased estimators with the same $O(1/D)$ variance.

Step 4 — Central result

Rahimi & Recht 2007 — key identity: $$k(\mathbf{x}, \mathbf{y}) = \mathbb{E}_{\substack{\boldsymbol{\omega}\sim p \\ b\sim U[0,2\pi]}} \!\left[ \underbrace{\sqrt{2}\cos(\boldsymbol{\omega}^\top\mathbf{x}+b)}_{z_\omega(\mathbf{x})} \cdot \underbrace{\sqrt{2}\cos(\boldsymbol{\omega}^\top\mathbf{y}+b)}_{z_\omega(\mathbf{y})} \right]$$

Step 5 — Monte Carlo with $D$ samples

$$z(\mathbf{x}) = \frac{1}{\sqrt{D}} \begin{bmatrix} \sqrt{2}\cos(\boldsymbol{\omega}_1^\top \mathbf{x} + b_1)\\ \vdots\\ \sqrt{2}\cos(\boldsymbol{\omega}_D^\top \mathbf{x} + b_D) \end{bmatrix}\!\!, \quad \boldsymbol{\omega}_j \sim p(\boldsymbol{\omega}),\; b_j \sim U[0,2\pi]$$ $$z(\mathbf{x})^\top z(\mathbf{y}) \approx k(\mathbf{x},\mathbf{y})$$

Pseudocode

# Given: kernel k, dataset X (n×d), D features
# Output: feature matrix Z (n×D)

# 1. Identify spectral density p(ω):
#    RBF     → p(ω) = N(0, σ⁻²I)
#    Laplace → p(ω) = Cauchy
#    Matérn  → p(ω) = Student-t

# 2. Sample random frequencies and phases
omega = sample(p_omega, shape=(d, D))  # (d, D)
b     = uniform(0, 2π,  shape=(D,))   # (D,)

# 3. Build feature map
Z = sqrt(2/D) * cos(X @ omega + b)    # (n, D)

# 4. Train any linear model on Z
#    ↳ approximates the kernel machine
model = LinearSVM().fit(Z, y)

# Predict new point x*:
z_star = sqrt(2/D) * cos(x_star @ omega + b)
y_pred = model.predict(z_star)

🔬 RBF & Spectral Self-Duality

Why is $p(\boldsymbol{\omega}) = \mathcal{N}(\mathbf{0}, \sigma^{-2}\mathbf{I})$ for the RBF kernel?

We need $p$ such that:

$$\exp\!\left(-\frac{\|\boldsymbol{\Delta}\|^2}{2\sigma^2}\right) = \int p(\boldsymbol{\omega})\, e^{i\boldsymbol{\omega}^\top\boldsymbol{\Delta}}\, d\boldsymbol{\omega}$$

This is a Fourier inversion problem. The key fact: the Gaussian is self-dual under the Fourier transform:

$$\mathcal{F}\left\{e^{-t^2/2\sigma^2}\right\}(\omega) = \sigma\sqrt{2\pi}\; e^{-\sigma^2\omega^2/2}$$

Normalizing to a probability density:

$$\boxed{p(\omega) = \mathcal{N}\!\left(0,\, \frac{1}{\sigma^2}\right)} \quad\Longrightarrow\quad \mathbb{E}_{\omega \sim \mathcal{N}(0,\sigma^{-2})}[\cos(\omega\Delta)] = e^{-\Delta^2/2\sigma^2}$$

In $d$ dimensions with the isotropic RBF kernel: $p(\boldsymbol{\omega}) = \mathcal{N}(\mathbf{0},\, \sigma^{-2}\mathbf{I}_d)$, equivalently $\boldsymbol{\omega} = \mathbf{z}/\sigma$, $\mathbf{z}\sim\mathcal{N}(\mathbf{0},\mathbf{I})$.

Large $\sigma$ (wide, smooth kernel)
→ Long-range correlations
→ Low-frequency structure
→ Spectral density narrow (small variance $\sigma^{-2}$)
Small $\sigma$ (narrow, sharp kernel)
→ Short-range correlations only
→ High-frequency structure
→ Spectral density wide (large variance $\sigma^{-2}$)

This is the time-frequency uncertainty principle: wide in space ↔ narrow in frequency.

📏 Error Guarantees

Unbiasedness

$$\mathbb{E}\!\left[z(\mathbf{x})^\top z(\mathbf{y})\right] = k(\mathbf{x},\mathbf{y})$$

No systematic error — only random noise that vanishes as $D \to \infty$.

Variance decay

$$\text{Var}\!\left[z(\mathbf{x})^\top z(\mathbf{y})\right] = O\!\left(\tfrac{1}{D}\right)$$

Variance halves every time you double $D$.

Pointwise concentration (Hoeffding)

Since $z_\omega(\mathbf{x}) \in [-\sqrt{2},\sqrt{2}]$, the product is bounded in $[-2,2]$:

$$\Pr\!\left[\,\left|z(\mathbf{x})^\top z(\mathbf{y}) - k(\mathbf{x},\mathbf{y})\right| \geq \varepsilon\,\right] \leq 2\exp\!\left(-\frac{D\varepsilon^2}{4}\right)$$

To achieve $|\text{error}| \leq \varepsilon$ with probability $\geq 1-\delta$:

$$D \geq \frac{4}{\varepsilon^2}\ln\frac{2}{\delta}$$
$D$ is independent of input dimension $d$ and dataset size $n$. This is the key practical advantage.

Uniform bound (Rahimi & Recht)

For all pairs on a compact domain $\mathcal{M}$:

$$D = O\!\left(\frac{d}{\varepsilon^2}\ln\frac{\sigma_p\,\text{diam}(\mathcal{M})}{\varepsilon\delta}\right)$$

For uniform accuracy, $D$ scales linearly with $d$.

🏛 Extensions & History

The Original Paper

"Random Features for Large-Scale Kernel Machines"
Ali Rahimi and Benjamin Recht, NeurIPS 2007

By 2007 kernel SVMs were the gold standard for classification. But training at $n > 10^5$ was computationally out of reach. Rahimi & Recht provided the first theoretically grounded, practically efficient solution: approximate the kernel function itself rather than trying to speed up the linear algebra. The paper won the NeurIPS 2017 Test of Time Award and has since accumulated thousands of citations spanning kernel approximation, GP regression, deep kernel learning, and attention in Transformers.

The "Alchemy" Talk (NeurIPS 2017)

Rahimi used his Test of Time acceptance speech to argue that ML had become "alchemy" — practitioners producing results through trial-and-error without understanding why things work.

"We are building systems that govern healthcare and mediate our civic dialogue... I would like to live in a society where systems are built on top of verifiable, rigorous, thorough knowledge and not alchemy."

He called for ablation studies, simple toy problems, and theorems that can be transmitted to the next generation. The talk provoked a public rebuttal from Yann LeCun, who argued that engineering advances without complete theoretical understanding is normal across all fields.
